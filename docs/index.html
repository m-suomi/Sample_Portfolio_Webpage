Denver Neighborhoods

House Value, Accidents, Crime, and
Public Transit by Neighborhood

We investigated Denver Neighborhoods for our first group project. We started off by pulling the Median House Price for each Neighborhood from Zillow's API (the image to the left shows relative median house prices by neighborhood - the darker the shade of the purple, the higher the median house price of the neighborhood). We then pulled in arrest and accident data from the Denver Police Department. Lastly, we pulled in traffic commute data from each neighborhood to four different common commute destinations from Google Directions API in order to determine how much slower commute times for both driving and public transit were during common rush hour times of the work week. Lastly, we performed some cool visualizations using the gmaps python library and the Google API. We imported a geocode json of the Denver neighborhood boundaries from City of Denver's webpage, normalized each data set we were graphing by neighborhood, and then mapped the normalized value as an opacity to the google map.

What's the Weather Like?

A Data Based Investigation of Latitude

What's the weather like as we approach the equator? In this example, we created a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, you'll be utilizing a simple Python library citipy, the OpenWeatherMap API, and a little common sense to create a representative model of weather across world cities. We built a series of scatter plots to showcase the following relationships: Temperature (F) vs. Latitude, Wind Speed (mph) vs. Latitude, Humidity (%) vs. Latitude, Cloudiness (%) vs. Latitude.
